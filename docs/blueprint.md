# **App Name**: GameNight

## Core Features:

- Game Play Interface: Interactive screen to display one game task at a time (Challenge, Never Have I Ever, Prompt) and allow players to advance.
- Game Selection & Lobby: Enable users to browse available pre-defined party games and start a new session.
- PWA Offline Capabilities: Implement service workers to cache essential game data, allowing selected games to be playable offline after initial access.
- Static Game Content Loading: Fetch and parse game rules and content from static JSON files embedded within or served to the application.
- Custom Game Code Input: Provide a mechanism for users to input a unique code to load a special custom game pre-defined by an administrator.
- Advertisement Integration: Display non-intrusive banner advertisements and infrequent fullscreen ads at natural breaks for monetization.

## Style Guidelines:

- Color scheme: Dark theme, reflecting an evening party atmosphere, ensuring elements pop against a subdued background.
- Primary color: Vibrant Red (#FA3838) for energy and urgency, akin to a party cup or an alert, making key actions and highlights stand out.
- Background color: Dark Charcoal (#1C1717), a very dark, slightly desaturated hue of the primary, providing a deep, immersive canvas.
- Accent color: Punchy Magenta (#FA38D6), an analogous hue that provides strong contrast and adds to the energetic party vibe, suitable for secondary actions or decorative elements.
- Font: 'Poppins' (sans-serif) for both headlines and body text. Its geometric, contemporary style provides a bold yet legible presence suitable for the dynamic and direct nature of game prompts.
- Icons: Utilize clear, stylized, and minimalistic icons related to party, social interaction, and common drinking game elements (e.g., dice, cards, glasses), complementing the bold visual style.
- Main game layout: Focused and minimalist, presenting 'one active screen' with a 'single task at a time' prominently displayed, minimizing distractions to facilitate social interaction and clear game progression.
- Animations: Subtle and smooth transitions between game tasks or screens to enhance the 'social flow' and provide satisfying visual feedback without interrupting gameplay.
